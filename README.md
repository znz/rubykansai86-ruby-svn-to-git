# Ruby svn to git

第86回 Ruby関西 勉強会での発表資料です。

Ruby本体の開発のメインリポジトリがsubversionからgitに変わったので、その話をします。
k0kubunさんのブログの[令和時代のRubyコア開発](https://k0kubun.hatenablog.com/entry/ruby-core-2019)を見ておくとわかりやすいかもしれません。(このブログ記事からの抜粋と関連する話をする予定です。)
